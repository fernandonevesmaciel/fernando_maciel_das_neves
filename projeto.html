PROJETOS

To-Do List: Organize suas Tarefas

A To-Do List é uma aplicação web simples, ágil e intuitiva, projetada para ajudar o usuário a organizar suas tarefas diárias com um clique. Basta digitar uma tarefa, adicioná-la à lista e, ao clicar sobre ela, marcá-la como concluída — tudo de forma imediata e visualmente satisfatória. Desenvolvida utili­zando apenas HTML, CSS e JavaScript, sem bibliotecas externas, a aplicação demonstra clareza no uso da lógica de programação e manipulação de DOM. A interface minimalista e funcional garante que qualquer usuário entenda a dinâmica rapidamente, propósito que reflete meu compromisso em criar soluções úteis e direto ao ponto.

Calculadora de Despesas Pessoais

A Calculadora de Despesas Pessoais é uma aplicação web criada para facilitar o controle financeiro do dia a dia de forma simples e intuitiva. O usuário pode inserir diferentes valores de gastos e obter imediatamente o cálculo automático do total, garantindo mais organização e clareza sobre suas finanças. Todo o projeto foi desenvolvido utilizando apenas HTML, CSS e JavaScript, sem bibliotecas externas, resultando em um sistema leve, responsivo e acessível em qualquer dispositivo.

Quiz Interativo

O Quiz Interativo é uma aplicação web dinâmica e envolvente projetada para testar conhecimentos gerais de forma divertida e interativa. Com uma interface simples e direta, o usuário é convidado a responder uma série de perguntas — avançando uma por uma — com a possibilidade de reiniciar o quiz a qualquer momento, o que garante uma experiência leve e repetível. Todo o projeto foi construído somente com HTML, CSS e JavaScript, demonstrando habilidade em manipulação de DOM, lógica de controle de fluxo e design responsivo. Este quiz não só reflete meu interesse por criar experiências interativas e engajantes, como também mostra meu foco em usabilidade e clareza visual.
